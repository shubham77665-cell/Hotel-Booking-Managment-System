# 🏨 Hotel Booking Management System

## 📌 Project Overview

The Hotel Booking Management System is a web application developed to simplify hotel operations by managing room bookings, customer records, and reservations. It provides a secure and efficient platform for hotel staff to manage bookings and customer information.

---

## 🎯 Features

- User Registration & Login
- Room Booking
- Booking Management
- Customer Management
- Room Availability
- Admin Dashboard
- Secure Authentication

---

## 🛠 Tech Stack

- Python
- Flask
- SQLAlchemy
- SQLite
- HTML
- CSS
- Bootstrap

---

## 🚀 How it Works

1. User registers or logs in.
2. Browse available rooms.
3. Book a room.
4. Admin manages reservations.
5. Database updates automatically.
